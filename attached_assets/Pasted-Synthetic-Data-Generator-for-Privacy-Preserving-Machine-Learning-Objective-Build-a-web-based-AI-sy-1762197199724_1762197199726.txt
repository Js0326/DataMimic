Synthetic Data Generator for Privacy-Preserving Machine Learning

Objective:
Build a web-based AI system that generates synthetic datasets preserving the statistical properties and ML utility of original datasets while ensuring data privacy and anonymity.

The system allows users to upload real-world CSV datasets (e.g., health, finance, demographics), choose a generation model, and obtain a synthetic dataset that mimics the real one without exposing sensitive records.
It provides visual analytics, privacy metrics, and downloadable outputs for easy use in ML training and research.
